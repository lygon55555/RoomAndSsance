# RoomAndSsance
room &amp; ssance

### How to run

1. Download Files

```jsx
git clone https://github.com/lygon55555/RoomAndSsance.git
```

2. Move to directory and Command execution

- You need Xcode to execute this project

```jsx
cd RoomAndSsance
pod install
open RoomAndSsance.xcworkspace
```

3. Build and Run
